===script===
    angular.directive('my:time', function() {
      return function($defer, linkElement) {
        (function updateTime(){
          linkElement.text(new Date().toString());
          $defer(updateTime, 1000);
        })()
      };
    });
    ===/script===
    

  
* We can teach HTML new tricks
* Directives can add behavior to any HTML element
* DI helps with testability
