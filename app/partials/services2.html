===script===
    function MyCntrl($resource){
      var Book = $resource('books.json');

      this.fetch = function(){
        this.books = Book.query();
      }
    }
    ===/script===
    

      
      
{{books}}

    

  

* $resource allows the for RESTfull API
* $resource separates the definition part form the usage
* Because it is injectable service it can be easily mocked/tested
* `query` is an example of operation, but many more can be defined
* callbacks are optional
